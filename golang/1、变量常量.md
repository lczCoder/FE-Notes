### go语言 常量和变量

1、定义变量关键词 var

默认定义方法

> var <变量名> <变量类型> = <值>

变量的类型要和值相匹配

```go
/* 基本数据类型 */
var a int = 0 // 整型
var b string = "string" // 字符型
var c bool = true // 布尔型
// 十进制下，float32的有效数字大约是6位，float64的有效数字大约是15位
var d float32 = 1.0 // 浮点型 32位
var e float64 = 1.000 // 浮点型 64位

/* 其他数据类型 */
var f byte = 0 // 字节
var g func() = func() {} // 函数
```
